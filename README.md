# 🚀 Python GUI Port Scanner (Multi-threaded)

A simple and fast **GUI-based Network Port Scanner** built using Python.  
This tool allows users to scan a target IP address or hostname and identify open ports in real-time.

---

## 📌 Features
- ✅ User-friendly GUI (Tkinter)
- ⚡ Multi-threaded fast scanning
- 📊 Real-time progress tracking
- 🔍 Detects open ports and common services
- 💾 Save scan results to file
- ❌ Stop scan anytime

---

## 🛠️ Technologies Used
- Python
- Socket Programming
- Tkinter (GUI)
- Multi-threading

---

## 🎯 How It Works
1. Enter target IP or hostname  
2. Select port range (e.g., 1–1024)  
3. Click **Start Scan**  
4. View open ports in real-time  
5. Save results if needed
